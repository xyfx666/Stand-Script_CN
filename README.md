# Stand-Script_CN

# 因为仓库内主要人员都已停止翻译，故暂时停更

## 一.我们是做什么的？  
我们会翻译一些Stand的LUA脚本，并且会不定期添加项目，每个翻译的脚本都会在此仓库实时更新，并在翻译完成后发布Releases

## 二.如何下载成品？
如果您只想要成品的话，您可以前往以下链接获取。  
https://github.com/xyfx666/Stand-Script_CN/releases

喜欢的话欢迎给我们点个Star

## 三.如何参加翻译？  
您只需在您翻译完后提交PR即可。

## 四.文件夹含义？

To be proofread 待校对/正在校对  
Completed 已完成  
Origin 原版文件(部分)
